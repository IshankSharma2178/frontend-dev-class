Responsive Design Principles

Fluid Layouts

Use relative units like %, fr, and minmax() to allow content to reflow.

.container { width: min(1100px, 92%); }

Flexible Media

Make images and videos scale within their containers.

img { max-width: 100%; height: auto; }

Breakpoints

Introduce layout changes at meaningful viewport widths.

@media (max-width: 700px) { … }
